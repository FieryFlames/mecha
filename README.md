# Mecha

A multipurpose Discord bot for [FortniteLeaks] using [sapphire framework][sapphire] written in TypeScript

### Prerequisite

```sh
npm install
```

### Development

```sh
npm run watch:start
```

[sapphire]: https://github.com/sapphiredev/framework
[fortniteleaks]: https://github.com/sapphiredev/framework
